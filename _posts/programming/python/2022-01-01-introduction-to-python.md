- Variables are containers for storing the data values, with which we can further reference and manipulate the values.
- Literals are values used in Python. Types are:
1. Numeric
    - Integer
    - Float
    - Complex 
2. Boolean
3. Collection
    - List
    - Tuples
    - Dictionary
    - Sets
4. String
5. Special
